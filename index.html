거대한 우주부터 미시 세계까지, 세상을 지배하는 하나의 암호

자기 유사성(Self-Similarity): 부분과 전체가 같은 모양으로 무한히 반복되는 성질.

자연은 가장 효율적인 방식으로 자신을 복제하며 세계를 구축합니다. 우리는 이 숨겨진 규칙을 '프랙탈(Fractal)'이라 부릅니다.

"자의 길이가 줄어들수록 유한한 해안선의 길이는 무한하게 늘어난다."
- 만델브로(Mandelbrot)

화학과 생명과학 : 분자 구조에 숨겨진 자연의 최적화

[생명과학] 남조류 구연산 합성 효소의 진화

  • 자연 상태에서 시어핀스키 삼각형 구조(레벨 1, 2)로 자가 조립.
  • 단순한 기하학이 아닌, 생화학적 피드백 조절과 효소 활성화를 위한 자연의 '진화적 선택'.

[화학] 나노 구조체, 덴드리머(Dendrimer)

  • 반복적인 확장을 통해 세대(Generation)를 거듭하며 성장하는 분자.
  • 약물 전달 및 정밀 센서에 활용되는 인공 프랙탈 화학의 정수.

유한한 공간 속 무한한 세상, 수학으로 통제하다

[시어핀스키 삼각형]

$n=1 \rightarrow n=2 \rightarrow n=3$

넓이는 0에 수렴하지만, 길이는 무한대가 되는 반복의 마법.

[코흐 눈송이]

$n=1 \rightarrow n=2 \rightarrow n=3$

변의 길이가 무한히 늘어나는 1.262 차원의 역설.

수학은 이 무한하고 복잡한 자연의 패턴을 알고리즘으로 분해하고,
코드로 통제할 수 있게 해주는 '결정적 언어'입니다.

관찰자를 넘어 창조주로 : 여러분의 미션

"작은 조각 속에 전체의 모습이 숨어 있는 신기한 패턴, 프랙탈.
우리 주변의 숨은 규칙을 AI와 함께 해석하고
인터랙티브하게 표현하여 모두와 공유합시다."

자연의 숨은 규칙을 수학·과학적으로 해석하고,
AI와 상상력을 더해, 코드로 환경을 통제하는 '나만의 새로운 생태계'를 만드세요.

프로젝트 수행 로드맵 (3단계 확장 모델)

1. 기본 탐구

(규칙의 발견)

발견된 반복적 규칙을 수학적/과학적으로 조사 및 분석.

2. 상상하기

(대안적 모델링)

"만약 환경이 바뀐다면?" 질문을 던지고 AI와 토론.

3. 표현하기

(인터랙티브 구현)

코드를 활용해 실시간으로 변수가 조작되는 시뮬레이션 앱 제작.

Step 1. 기본 탐구 : 규칙의 알고리즘화

[수학 + 생물]

L-System과 식물의 분기 구조.

가지치기 규칙이 햇빛 수용 효율을 어떻게 극대화하는가?

[수학 + 물리]

DLA(확산 제한 응집)와 리히텐베르크 도형.

번개가 저항이 적은 최적의 경로를 찾아가는 원리.

[수학 + 화학]

프랙탈 차원(1.262)과 탄화수소(알케인) 구조 분석.

유기 화합물의 지그재그 선형 구조 복잡도 도출.

Step 2. 상상하기 : 질문과 AI의 결합

자연의 기존 규칙을 벗어나, 환경 변수를 비틀어보는 융합적 사고를 시작하세요.

Q. 만약 이 식물(L-System)이 중력이 전혀 없는 우주나, 빛이 한 방향에서만 극단적으로 비추는 행성에서 자란다면 형태가 어떻게 바뀔까?

Q. 만약 번개(DLA)가 공기 저항이 아닌 소음 에너지의 밀도를 따라 방전되도록 설계한다면 어떤 시각적 패턴이 나올까?

AI (Claude, Gemini 등)를 아이디어 발상 파트너로 활용하여 논리적인 대안 환경 모델을 구축하세요.

Step 3. 표현하기 : 관찰을 넘어 '조작'의 영역으로

[기존의 산출물]

  • 관람객의 역할: 수동적 관찰자 / 감상
  • 산출물 형태: 정적인 프랙탈 그림, 발표용 PPT
  • 결과물: 고정된 하나의 결과값

[우리의 목표 산출물]

  • 관람객의 역할: 능동적 참여자 / 창조주
  • 산출물 형태: 환경 변수를 통제하는 인터랙티브 웹앱/파이게임
  • 결과물: 변수(슬라이더) 조작에 따라 실시간으로 변화하는 무한한 결과값

결과를 단순히 보여주는 것에 그치지 마세요. 관람객이 직접 통제하게 만드세요.

[산출물 예시 1] 차원 조작기 (수학 + 정보)

유한한 면적에서 무한한 길이가 생성되는 수학적 역설을 실시간으로 조작하는 웹앱.

Control Panel (UI 변수)

  • 성장 세대(n) 조절: 1세대 ~ 10세대 슬라이더
  • 분할 각도 조절: 60도 ~ 120도 슬라이더
  • 넓이 색상 채우기: On/Off 토글 버튼

→ 관람객이 n세대를 직접 늘려가며 프랙탈 차원의 복잡성을 직관적으로 체험.

[산출물 예시 2] 디지털 바이오 생태계 시뮬레이터 (생명 + 정보)

빛과 중력 등 대안적 환경 변수에 따라 식물의 군락이 어떻게 적응해 변모하는지 관찰하는 시뮬레이터.

>>> L-System String: F -> FF+[+F-F-F]-[-F+F+F]

  • 중력의 크기: (0.1g ↔ 2.0g)
  • 조명 컨트롤: (빛의 방향 및 세기: 0도 ↔ 180도)
  • 생존 변수: (가지치기 확률 / 돌연변이 발생률)

[산출물 예시 3] 혼돈 속 에너지 방전 모델 (화학/물리 + 정보)

리히텐베르크 도형 및 수소 연소 최적화 경로 알고리즘(DLA)을 활용. 마우스 클릭에 따라 에너지가 저항이 가장 적은 경로를 폭발적으로 찾아가는 인터랙티브 아트.

  • [매핑 버튼] 가상 공간의 '저항 맵(Obstacles)' 실시간 그리기
  • [수소 농도 및 에너지 응집도 조절] Low ↔ High 슬라이더
  • [색상 토글] 방전 속도에 따른 열에너지 스펙트럼 시각화 (Cold ↔ Hot)

나만의 생태계를 창조할 디지털 무기고 (Toolstack)

[아이디어 & 로직]

LLM (Claude, Gemini 등)

복잡한 수학식(망델브로 차원 등)과 파이썬 기본 뼈대 코드(Boilerplate) 설계.

[구현 & 시뮬레이션]

JavaScript (p5.js), Python (Pygame)

웹 브라우저나 독립 실행형 앱으로 실시간 렌더링 및 UI 슬라이더 구현.

[사운드 융합]

Suno (음악창작 AI)

프랙탈의 기하학적 느낌을 담은 BGM 생성.

[발표 & 미디어]

Typecast (TTS AI), Canva

앱 시연 영상을 위한 도슨트 및 UI 디자인 제작.

완벽한 계획서 작성을 위한 4가지 체크리스트

  • [ ] 1. 융합의 명확성: 단순한 프랙탈 조사가 아닌, 교과 지식(물리, 화학, 생명 등)과 정보(코딩)가 융합된 뚜렷한 주제인가?
  • [ ] 2. 핵심 수식/알고리즘: 탐구의 뼈대가 될 수학/과학적 알고리즘이 구체적으로 명시되었는가? (예: L-System, DLA 알고리즘 등)
  • [ ] 3. 환경 변수 통제: 내가 상상한 세계에서 조작할 '환경 변수'가 명확한가? (예: 빛의 양, 중력, n세대 변화 등)
  • [ ] 4. 인터랙티브 요소: 관람객이 단순히 보는 것을 넘어, 직접 변수를 조작하고 즉각적인 피드백을 경험할 수 있는 UI 요소가 포함되어 있는가?
